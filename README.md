# Yash Jain Portfolio

Production-ready Next.js academic portfolio for Yash Jain.

## Stack
- Next.js App Router + TypeScript (strict)
- Tailwind CSS with centralized design tokens
- MDX-driven projects/blog routes
- Vercel Analytics

## Structure
- `app/` route files and metadata
- `components/` reusable section components
- `content/` centralized profile and MDX content
- `lib/` SEO helpers, utilities, project metadata
- `styles/tokens.css` theme tokens

## Development
```bash
npm install
npm run dev
```

## Deployment
Push to GitHub and connect the repository to Vercel for automatic deployments.
